# /handoff

사용법: `/handoff <주제>`

`journal/YYYY-MM-DD-{주제}-handoff.md` 생성.

## 포함 내용

1. **작업 배경** — 왜 이 작업을 했나, 관련 Jira 이슈
2. **완료 사항** — 구체적으로 무엇이 됐나
3. **미완 사항** — 무엇이 남았나, 어디서 막혔나
4. **다음 할 일** — 순서대로 구체적으로
5. **주의사항** — 함정, 미확인 사항, 확인 필요한 것
6. **관련 파일** — 핵심 파일 경로 목록
